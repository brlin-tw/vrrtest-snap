# Unofficial Snap Packaging for VRRTest

![(Placeholder) Icon of VRRTest](https://cdn.statically.io/gl/brlin/vrrtest-snap/699d301/snap/gui/placeholder-icon.png "(Placeholder) Icon of VRRTest")

**This is the unofficial snap for VRRTest**, *"A small utility to test Variable Refresh Rate(VRR) capable displays on Linux."*. It works on Ubuntu, Fedora, Debian, and other major Linux distributions.

[![Status Badge of the `vrrtest` Snap](https://snapcraft.io/vrrtest/badge.svg)](https://snapcraft.io/vrrtest)

## Installation

([Don't have snapd installed?](https://snapcraft.io/docs/core/install))

### In a Terminal

    # Install the snap #
    sudo snap install vrrtest

    # Launch the application #
    vrrtest

### The Graphical Way

[![Get it from the Snap Store](https://snapcraft.io/static/images/badges/en/snap-store-black.svg)](https://snapcraft.io/vrrtest)
